## Installation

### Swift Package Manager

```swift
dependencies: [
    .package(url: "https://github.com/liwpk/MySuperExtension.git", from: "1.0.0")
]
```

Add to your target:

```swift
.target(name: "YourApp", dependencies: ["MySuperExtension"])
```

### Xcode

File → Add Package Dependencies → `https://github.com/liwpk/MySuperExtension.git`
